Epic Games account

How to make game portable

ScaraboGhoul game has been developed with Epic Online Service to make it portable. To achieve that effect, user has to create an account on Epic Games (EG) service. In the main menu, click Sign In button.

Task controller will launch the Epic Games website to check valid account on the user's default web browser. If that is the first time to sign in, the website looks like the image below:

Create an Epic Games account

The next step is to click the Sign Up button, Epic Games server will asks user to choose to sign in another social media account, such as Gmail account, Facebook account, etc., from there to create a new Epic Games account for user. It's recommended to sign up with an email address - the first option on the panel. That social media account will be used to validate more user identification in the future, so it's good to put it down with the new Epic Games account somewhere like a notebook, diary, sticker, or digital note app on the phone preventing from creating another EG account with another social media account if the user forgets their last social media account used to register EG account.

After user signs up successfully, it will show a dialog box like image below, click Continue To App button to continue account verification:

Then, click Allow button to finish sign-in process, see image below:

Successful sign-in

If this sign-in process is successful, this verification tab will be closed on the web browser automatically. In addition, when user get back to game app, there are two other hints to indicate a successful sign-in process. User must see the green statement Your identity is clear. Welcome! on the top left corner, and an online chat box on the top right. See the image below:

The next time user launch game and click Sign In button, it will jump directly to the EG account verification website, not the SignUp one. However, once in a while, it will launch the SignUp website again to assert user identity for a long-term security.

This EG account will store over half of user's game data, save a decent amount of fee to store date on blockchain. Additionally, due to its saving on an online storage, user can continue to play game where they left off on any desktop having this game downloaded and a verified EG account, which makes it portable.

Pay attention, user EG account can only be valid on a desktop at a time, that means user can only play game on one desktop because the EG account verified on the last desktop disappears.

Next, click Game button to move to wallet account interface.

note

Scaraboghoul game is an online game, continuously connecting with blockchain and EG service to send and get data - make sure to have it connected with a stable internet, any disconnection in a decent amount of time can crash the game and leave some game data unsaved.
