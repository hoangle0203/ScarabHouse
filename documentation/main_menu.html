Main menu

Main menu buttons

After the game loads successfully, the first menu appearing is the main menu. Users can click:

  • Exit button to quit game.
  • Preview button to see game in advance at the hatching stage.

Preview game interface

In preview game interface, user can click:

  • Double nest button to switch to Double nest mode.
  • Single nest button to switch to Single nest mode.
  • Male and Female buttons to see and select the gender and the shape of Scarab egg.
  • The left and right color panels are used to set color for Scarab egg on the left and right side respectively.
  • In the color panel, most of colors can be applied for both gender, there are only two colors set aside for a specific gender of Scarab egg which includes, in the form of male - female, amber - citrine and tanzanite - tourmaline.
  • Buy NFT button to jump to OpenSea website buy your favorite Scarab egg(s) after visiting Preview Game scene.
  • Back button to back to main menu.
