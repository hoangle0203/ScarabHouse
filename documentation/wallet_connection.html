Wallet connection

Add wallet account

ScaraboGhoul game is a blockchain game, so it needs a valid crypto-based wallet account to process gamer's data. In the wallet account interface, click the wallet account list, select Add Wallet Account option, then fill the box below with private key of that wallet.

Then, click Add Account button to start wallet account generation.

When user click the Add Account button, embedded code behind will check to see if it is a valid wallet account. So how to make it valid? The target blockchain this game has been developed for is Polygon Mainnet, so wallet address must be a valid Ethereum address. The address must be 40-hexadecimal-character address and private key must be 64-hexadecimal-character string. Any invalid private key will cause the Ethereum address generation to stop. For example, if the number of private key characters is not 64, user will see a prompt like image below:

Or any character in private key string is not a hexadecimal character:

If private key is valid, the Ethereum address generation will process and finish almost immediately, and user can check that completion by clicking the wallet account list, the account just added sits neatly in the list.

This Eth address generation is processed offline and in a secure manner, the generated address will be saved on data storage along with user's EG account, so next time playing game, it's not necessary to add again because it's already in the list.

It's possible to add more than one Eth address to the list, but to move deeply to the game and process blockchain data, one address must be chosen. After the desired address selection, there are two other buttons appearing. User can click Enter button to move to purchase interface, click Remove Account button to remove the selected wallet account, or click Back button to get back to main menu.

How to use in-game wallet

User can open wallet interface by clicking the button having a wallet image in any interface including that wallet button. For example, in the purchase interface as image below, click Wallet button to open wallet interface.

In the wallet interface, user can acknowledge which wallet account is used to process all blockchain transactions in regular or in game. For regular ones, there are currently five networks supported to send only crypto to another wallet including Ethereum, Polygon, Avalanche, Binance and Cronos Mainnets. And there is no option to deposit or withdraw cash in this wallet scope from those networks.

After choosing desired network, user must fill receiver's address and the amount of crypto needed to send. Max button can used to send all the current amount of crypto in a fast way, but do consider it deliberately before moving on. Then, click Send button to open the order interface to have a quick check over the transaction details, or click Clear button to clear all transaction information.

note

For transactions in game, Polygon Mainnet must be chose before game actions (buy, sell eggs, etc.); if not, that transaction will fail and a message appears to prompt user to switch to the proper network.
