Downloading Game

How to play

Go to home page, in the left Scarab token, choose the proper button to download the game package (in .zip format) according to your PC platform. ScaraboGhoul game is currently only played on desktops running Windows OS and Mac device.

After downloading the game package, unzip it, double click .exe file to play game.

note

For the first time this game runs, your security system may pop up a dialog box to ask for network access's permission. This image below shows that dialog box in Windows system, just check the Public networks option and click Allow Access button to move on.

Desktop specs requirements

ScaraboGhoul game is a 3D game rendering quite a lot of 3D objects and textures, it needs some decent graphical specifications from user's desktop hardware to run smoothly.

Recommended PC specs:

  • CPU: 8-core CPU at 2.6 GHz.
  • RAM: 8 GB RAM.
  • GPU: Nvidia GTX 1650/ AMD Radeon 4700GE or higher.

Minimal PC specs:

  • CPU: 4-core CPU at 2.1 GHz.
  • RAM: 4 GB RAM.
  • GPU: Nvidia GTX 1080/ AMD RX Vega 64 or higher.
