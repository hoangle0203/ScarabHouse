Item managing interface

How to manage items in user's wallet

This section will introduce the interface of managing game items, it contains a few descriptive and wieldy buttons to process functionalities such selling, hatching, etc. Let's have a look at the image below.

Notes in the image above describe how to use game buttons in a clear way. It's still necessary to emphasize one thing that tokens hold in each user's wallet is divided into two lists - available list and sale list. Only tokens in available list are used to display objects and play in game level which prevents from the conflict that sale token is taken away while being used to play in game. In the image below, the selected token in the available list on the left side shows a Scarab egg shape and its simple properties while the selected token in the sale list on the right side doesn't show anything in the game level.

If user would like to see their for-sale tokens in the game item store in the purchase interface, just click the Back button at the right bottom corner switching to in-game item store interface.

There is an overhead widget above the object of selected token in available list for user to check over necessary details before making change to it.

How to work on game items

When a token is selected to sell in available list or to adjust the price in sale list, another box will overwrite the respective box for user to fill a price needed to complete the procedure.

The wallet interface will open and show the order for user to have a check-over the transaction summary before sending to process. User can click Confirm button to submit or click Cancel button to cancel the order.

For selling procedure, after its transaction gets through, that selected token in available list will be brought to sale list, user can open their sale list and check. Switching back and open the game item store in the purchase interface, the token just brought for sale will put at the bottom and its ownership now is On Sale.

If the owner of on-sale token put it back from game item store or it's been bought by another user, its ownership will change to Owned.
