In-game item store interface

Advantages of in-game item store

Purchasing NFTs directly in game item store is recommended to save a little gas fee compared to the surcharge users has to pay when purchasing on the 3rd party's application on web browsers. In this very manner, user only needs to pay gas fee that is necessary for blockchain to process game data, it's sure that there is no surcharge for everyone, even the game creator or contract owner - just pay what you need.

Another advantage of the item store in game is that user can see data of the item they want to buy in details, such as how many pieces of dung of four types the sale Scarab egg has consumed already, the royalty of each item, etc. before making a decision on the desired item's purchase.

How in-game item store works

In the purchase interface, click Buy NFT button to open the in-game store:

Select desired item, click Buy button open wallet interface displaying the order details, or click Back button to close the store. It's very important to set the network in Polygon Mainnet first before clicking the Buy button.

When the wallet interface put up to show an order from in-game item store, almost functionality will be frozen, and there are only two buttons in wallet being abled for user to work. Click Confirm button to send the order, or Cancel button to cancel the order:

Before hitting Confirm button, make sure to ownership status of the desired item is On Sale, if not, there is no order transmitted to the wallet and a message will be sent on the top left to show the problem. See image below:

note

With an aim to ease the time for transaction got through in queue, and for blockchain to process game data, user is begged to wait 10 seconds after that transaction before getting back to game.
